This is a simple and responsive Calculator Web Application that performs basic arithmetic operations.
The project is built using clean UI design and JavaScript logic to create a fully functional calculator.

FEATURES:
. Addition, Subtraction, Multiplication,  Division
.Decimal number support
.Clear (C) button to reset the display
.Delete last character functionality
.Instant calculation using "=" button
.Dark theme UI (modern look)
.Centered and responsive layout

 Tech Stack
.HTML
.CSS (Flexbox + Grid)
.JavaScript
